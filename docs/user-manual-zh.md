# 用户使用手册

本文说明 Go 合并版 Map Tile Fetcher 的实际使用方式。

## 1. 启动应用

进入 Go 应用目录：

```powershell
cd apps/admin-region-tiler
go run .
```

浏览器打开 `http://127.0.0.1:8081/`。

本地默认登录账号：

- 用户名：`admin`
- 密码：`adminmap`

可信本地环境可以通过 `AUTH_ENABLED=false` 关闭登录。部署环境建议保持登录启用，并通过本地环境变量或部署密钥覆盖默认用户名和密码。

## 2. 服务凭证

页面中的 **服务凭证** 区域同时用于行政区划下载和范围框选下载。

- **天地图 Token**：替换天地图源 URL 中的 `YOUR_TIANDITU_TOKEN`，选择天地图预览底图时也使用该字段。
- **Mapbox Token**：替换 Mapbox 源 URL 中的 `YOUR_MAPBOX_TOKEN`。
- **Mapbox SKU**：填写后替换 `YOUR_MAPBOX_SKU`；不填写时会移除该 SKU 查询参数。

凭证只随本次任务创建请求提交。不要把真实 token、`.env`、运行数据库、下载瓦片或生成压缩包提交到 Git。

## 3. 行政区划下载

当目标区域已有维护好的 GeoJSON 区域文件时，使用该模式。

1. 选择 **行政区划下载**。
2. 输入任务名称。
3. 在服务凭证区填写所选地图源需要的 token。
4. 选择地图源提供方，并勾选一个或多个图层。
5. 配置行政区划级别。默认从全球、中国开始，再按已维护数据选择省、市、区县。
6. 设置下载线程、保存线程、请求间隔、产物格式和执行时间。
7. 点击创建任务，并确认弹窗中的任务摘要。

系统会创建一个父任务，并按所选图层创建子任务。每个图层作为同一父任务下的一个子任务运行。

## 4. 范围框选下载

当目标区域需要在地图上直接选择时，使用该模式。

1. 选择 **范围框选下载**。
2. 输入任务名称。
3. 在公共服务凭证区填写 **天地图 Token**。
4. 选择天地图图层：`img`、`cia`、`vec`。
5. 选择矩形或多边形绘制模式。
6. 在地图上点击选择范围：
   - 矩形：点击两个对角点。
   - 多边形：至少点击三个点。
7. 确认或手动调整坐标和层级。
8. 查看瓦片估算、产物格式和执行时间。
9. 点击创建任务，并确认弹窗中的任务摘要。

范围坐标统一使用 WGS84 经纬度，瓦片方案使用 Web Mercator。

## 5. 计划任务

每个任务支持两种执行方式：

- **立即执行**：创建后立即进入队列。
- **单次定时**：到达选择的本地时间后执行。

定时任务在执行前会显示为 `scheduled` 或 `pending`。

## 6. 产物格式

创建任务时可以选择：

- **ZIP 文件树**：瓦片先写入文件系统，再打包为 ZIP 产物。
- **MBTiles**：直接生成 MBTiles，并作为产物下载。

产物保存在文件系统中，并由 SQLite 记录索引。SQLite 只保存控制数据和索引，不保存原始瓦片内容。

## 7. 任务管理

任务列表展示状态、进度、子任务、失败数量和产物状态。

可用操作取决于任务状态：

- 暂停运行中任务。
- 恢复已暂停任务。
- 取消计划中、等待中、运行中或已暂停任务。
- 删除已完成、失败、部分失败或已取消任务。
- 产物 ready 后下载产物。
- 存在可重试失败记录时，重试失败瓦片。

删除任务会清理任务记录和生成的任务产物。共享 GeoJSON 区域资源不会被删除。

## 8. 失败记录与重试

瓦片下载失败时，系统会持久化可重试失败记录，并在任务响应和失败记录接口中展示。

当任务菜单中出现 **重试失败瓦片** 时，可以只基于失败记录发起一次重试运行，不会重建完整任务。

常见失败类型：

- `429`：请求频率过高。
- `418` 或 blocked 分类：当前出口 IP 可能被地图源拦截。
- `502`、`503`、`504`：上游服务不稳定。
- proxy 或 network 分类：本地网络或代理链路异常。

出现限流或拦截时，建议增大请求间隔、降低线程数，或更换出口/代理。

## 9. 本地文件与安全

以下内容不得进入 Git：

- `.env`
- `data/`
- `output/`
- `tiles/`
- `bin/`
- `obj/`
- `publish*/`
- 日志、数据库、压缩包、二进制、截图和下载瓦片

文档和示例中允许保留的占位符：

- `YOUR_TIANDITU_TOKEN`
- `YOUR_MAPBOX_TOKEN`
- `YOUR_MAPBOX_SKU`

## 10. 维护者验证

发布或交付前执行：

```powershell
cd apps/admin-region-tiler
node .\scripts\release_preflight.mjs
```

发布预检会运行 Go 测试、前端语法检查、浏览器 UI 冒烟、敏感信息扫描和生成物扫描。
